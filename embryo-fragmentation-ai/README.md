# Automated Embryo Fragmentation Analysis Using AI

## Overview
This project explores an AI-assisted pipeline for analyzing embryo fragmentation from microscopic embryo images.

The goal of the system is to assist embryologists in assessing embryo quality by providing consistent and automated fragmentation analysis.

Due to clinical data privacy and NDA restrictions, the dataset and source code cannot be publicly shared. This repository provides a high-level overview of the system architecture and methodology.

## Problem Statement
Embryo quality assessment is an important step in assisted reproductive technologies. Fragmentation of embryos is one of the key indicators used by embryologists during evaluation.

Manual assessment can introduce variability and subjectivity. The objective of this project was to explore an AI-based approach for analyzing embryo fragmentation using image analysis techniques.

## System Architecture
The project pipeline includes the following stages:
1. Microscopy image acquisition
2. Image preprocessing and normalization
3. Region of interest extraction
4. Deep learning model inference
5. Fragmentation detection and scoring
6. Output visualization for embryologist review
The architecture diagram can be found in the `architecture` folder.

## Ethical and Privacy Considerations
This project was conducted using confidential medical data.
To respect data privacy and NDA agreements:
- No dataset is included
- No source code is shared
- No trained models are provided
This repository only documents the methodology and system design.
